# sao-typescript-boilerplate

> Basic typescript project

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao typescript-boilerplate my-project
```

### From git

```bash
sao edgardleal/sao-typescript-boilerplate my-project
```
